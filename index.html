Utils

Collection of web-based utilities for everyday tasks.

Available Tools

Discord Message Splitter

Split long messages into Discord-compatible chunks with smart newline handling.

  • Respects 2000/4000 character limits
  • Breaks at newlines when possible
  • Collapsible preview mode
  • One-click copy to clipboard
Regex Intersection Checker

Find the smallest word matching two regular expressions using NFAs.

  • NFA simulation with Thompson's construction
  • Pair comparison and bulk uniqueness checking
  • SVG-based NFA visualization
  • Supports common regex operators (*, +, ?, |)
